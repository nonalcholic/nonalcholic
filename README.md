# NONALCHOLIC

A small and fun KAIST personality type test project

## Development

### Database
```bash
cd path/to/repository
cd database
docker-compose up
```

### Backend
```bash
cd path/to/repository
cd server
docker-compose build
docker-compose up
```

### Frontend

```bash
cd path/to/repository
cd client
docker-compose build
docker-compose up
```
### Database + Backend + Frontend
```bash
cd path/to/repository
docker-compose build
docker-compose up
```
